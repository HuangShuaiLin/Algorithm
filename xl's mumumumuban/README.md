## 使用方式

每个模板文件均可独立编译使用，文件中包含：

- **结构体/函数封装** — 直接复制到代码中即可调用
- **注释说明** — 标注了时间复杂度、适用条件和典型场景
- **多版本** — 部分模板提供多个变体（如单哈希/双哈希、Prim/Kruskal）

### 示例

```cpp
// 使用线段树模板
#include "数据结构/线段树.cpp"

int main() {
    vector<int> a = {1, 2, 3, 4, 5};
    SegTree seg(a);

    seg.update(0, 2, 10);       // a[0..2] += 10
    cout << seg.query(0, 4);    // 查询区间和
}
```

## 维护说明

- 所有模板使用 **C++17** 语法（`auto`、结构化绑定、`string_view` 等）
- 默认使用 `1-indexed` 作为节点/数组索引（计算几何除外）
- 模板间尽量保持独立，减少交叉依赖
- 已有模板基于常见的竞赛需求，欢迎补充更多算法
